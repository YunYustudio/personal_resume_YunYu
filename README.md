# 刘峰宇在线作品集

这是一个纯静态单页作品集，可以直接打开 `index.html` 预览，也可以把整个 `online-portfolio` 目录上传到 Gitee Pages、GitHub Pages、对象存储或任意静态网站托管服务。

当前版本已支持：

- 中英文切换，语言状态会保存在浏览器本地
- 面向海外面试官的英文项目叙述
- 图片背景、樱花飘落、科技粒子、扫描线、滚动入场和截图大图预览
- 响应式布局与移动端导航优化

目录说明：

- `index.html`：页面内容与结构
- `styles.css`：响应式样式
- `script.js`：截图筛选与大图预览
- `assets/images/`：由原始项目截图生成的 WebP 网页资产

原始截图没有被修改，网站只使用生成后的副本。
